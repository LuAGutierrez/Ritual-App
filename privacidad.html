Ritual Volver

Política de privacidad

Última actualización: 15/03/2026

1. Datos que recopilamos

Podemos recopilar datos de registro (como email), información técnica de uso y datos necesarios para gestionar suscripciones y acceso.

2. Para qué usamos los datos

Usamos la información para crear y administrar cuentas, habilitar funcionalidades, procesar pagos y mejorar la experiencia del producto.

3. Base legal y consentimiento

El tratamiento se basa en la ejecución del servicio y en el consentimiento que otorgás al registrarte y aceptar términos y privacidad.

4. Servicios de terceros

Podemos usar proveedores externos para autenticación, base de datos y pagos. Cada proveedor procesa datos bajo sus propias políticas.

5. Conservación y seguridad

Aplicamos medidas razonables de seguridad. Conservamos los datos el tiempo necesario para prestar el servicio y cumplir obligaciones legales.

6. Tus derechos

Podés solicitar acceso, rectificación o eliminación de tus datos escribiendo a soporte@ritual.app.
